Audio Streaming with WebRTC

What does this do?

This demonstrates an audio streaming using WebRTC between the two browsers. Instead of using microphones, server side browser reads a MP3 file then streams it out to the client using WebRTC, while the client plays the audio stream in real-time.

How to run?

Click the links below to open server tab and client tab. By design, the client will become the initiator (generates "offer").

In the client page, you will see the first "offer" message in the text box. Copy the text to the sever side input box, then hit "OK". You will see "answer" message appears in the text box below. Copy the text, and paste it into the client side input box.

Repeat the above, until it stops showing a new text. On success, you should start hearing a sound played on the client side browser.
