Audio Streaming with WebRTC

What does this do?

This demonstrates a audio transmission using WebRTC between the two browsers. Instead of using micrphones, server side browser reads a MP3 file, trannsfer it to the client using WebRTC, then the client playbacks the audio.

How to run?

Click the links below to open server tab and client tab. By design, the client will become the oferrer (initiator).

In the client page, you will see the first "offer" message in the text box. Copy the text to the sever side input box, then hit "OK". You will see "answer" message appears in the text box below. Copy the text then paste it to the client side input box.

Reeapt the above, until it stops showing a new text. On success, you should start hearing a sound played on the client side browser.
